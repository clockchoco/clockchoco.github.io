# 한국사능력검정시험 기출 시대별 자동채점 웹사이트

## 실행

1. `index.html`을 브라우저에서 엽니다.
2. 컴퓨터에서는 왼쪽 문항 목록과 오른쪽 풀이 화면이 함께 보입니다.
3. 휴대폰에서는 목록에서 문항을 누르면 풀이 화면으로 이동합니다.

## 포함 범위

- 총 40개 문제 PDF
- 고급 제26~34회: 450문항
- 심화 제47~77회: 1,550문항
- 전체 2,000문항
- 공개 정답표 기반 정답·배점 2,000개 적용

## 기능

- 시대파트별 필터
- 급수·회차·상태·검색 필터
- 문항 이미지 보기 / 전체 페이지 보기
- 1~5번 선택 즉시 자동채점
- 맞은 문항 / 틀린 문항 필터
- 누적 점수 계산
- 즐겨찾기
- 메모
- 랜덤 20문항
- 진도 내보내기 / 가져오기

## 분류 및 정답 기준

텍스트 추출이 가능한 PDF는 문항 텍스트 키워드와 문항 순서를 함께 사용했습니다. 이미지형 PDF는 회차별 문항 순서를 기준으로 시대파트를 자동 배정했습니다.

정답과 배점은 공개 정답표 PDF를 파싱해 문항별로 적용했습니다. 모든 회차는 50문항, 회차별 총점 100점 기준으로 검증했습니다.
