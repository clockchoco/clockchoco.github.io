# 한국사능력검정시험 기출 시대별 자동채점 웹사이트

## 실행

1. `index.html`을 브라우저에서 엽니다.
2. 컴퓨터에서는 왼쪽 문항 목록과 오른쪽 풀이 화면이 함께 보입니다.
3. 휴대폰에서는 목록에서 문항을 누르면 풀이 화면으로 이동합니다.

## 포함 범위

- 총 40개 문제 PDF
- 고급 제26~34회: 450문항
- 심화 제47~77회: 1,550문항
- 전체 2,000문항
- 공개 정답표 기반 정답·배점 2,000개 적용

## 기능

- 시대파트별 필터
- 급수·회차·상태·검색 필터
- 문항 이미지 보기 / 전체 페이지 보기
- 1~5번 선택 즉시 자동채점
- 맞은 문항 / 틀린 문항 필터
- 누적 점수 계산
- 현재 조건 기준 완료율·정답률·남은 문항 요약
- 즐겨찾기
- 메모
- 랜덤·미풀이·복습 세트 출제
- 추출 텍스트 확인 및 다음 미풀이 이동
- 현재 필터 또는 전체 문항의 채점현황 일괄해제
- 진도 내보내기 / 가져오기
- GitHub Gist 기반 기기 간 진행도 동기화

## 기기 동기화

GitHub Pages는 정적 호스팅이라 웹페이지가 저장소 파일을 직접 수정할 수 없습니다. 대신 앱의 `기기 동기화` 메뉴에서 GitHub Gist JSON 파일을 진행도 저장소로 사용합니다.

1. GitHub 개인 액세스 토큰을 만들고 `gist` 권한을 부여합니다.
2. 웹페이지의 `기기 동기화` 메뉴에 토큰을 입력합니다.
3. 첫 기기에서는 `새 파일 만들기`를 눌러 Gist ID를 생성합니다.
4. 다른 기기에서는 같은 토큰과 Gist ID를 입력하고 `불러오기`를 누릅니다.
5. `자동 동기화`를 켜면 선택·메모·즐겨찾기 변경 후 자동으로 저장됩니다.

토큰은 저장소나 코드에 기록되지 않고 각 브라우저의 localStorage에만 저장됩니다. 공용 컴퓨터에서는 `설정 지우기`를 누르세요.

## 분류 및 정답 기준

텍스트 추출이 가능한 PDF는 문항 텍스트 키워드와 문항 순서를 함께 사용했습니다. 이미지형 PDF는 회차별 문항 순서를 기준으로 시대파트를 자동 배정했습니다.

정답과 배점은 공개 정답표 PDF를 파싱해 문항별로 적용했습니다. 모든 회차는 50문항, 회차별 총점 100점 기준으로 검증했습니다.
